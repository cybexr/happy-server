# Social Module

This module handles social interactions and user relationship management within the Happy Server application. It provides functionalities for managing friendships, user notifications, and user profile updates.

## Files Overview

*   **`friendAdd.ts`**: Contains logic for adding a new friend relationship between users.
*   **`friendList.ts`**: Provides functionality to retrieve a list of a user's friends.
*   **`friendNotification.spec.ts`**: Unit tests for the friend notification logic.
*   **`friendNotification.ts`**: Manages notifications related to friend requests and updates.
*   **`friendRemove.ts`**: Implements the functionality to remove an existing friend relationship.
*   **`relationshipGet.ts`**: Retrieves the relationship status between two users.
*   **`relationshipSet.ts`**: Allows setting or updating the relationship status between users.
*   **`type.ts`**: Defines shared types and interfaces used across the social module.
*   **`usernameUpdate.ts`**: Handles updating a user's username and broadcasting the change.
